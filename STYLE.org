* Purpose
This file defines the Rosé Pine theme target for LLM-assisted theme conversion.
Use it when converting existing themes, configs, CSS, editor themes, or UI token sets into Rosé Pine.

* Rules
:PROPERTIES:
:variant: moon
:output: full-file
:style: semantic-mapping
:END:

- Preserve functionality. Only change theme-related values unless a tiny structural change is required for clean theming.
- Prefer semantic Rosé Pine role mapping over blind hex replacement.
- Centralize repeated colors into variables or tokens when the target format allows it.
- Keep contrast readable and hierarchy clear.
- Preserve comments when reasonable.
- Be decisive. Do not leave placeholders.
- Default to =Rosé Pine Moon= unless another variant is explicitly requested.

* Semantic Roles
These roles come from the Rosé Pine palette guidance and should be used semantically rather than by nearest-color matching. [web:175][web:177]

- =base=: primary background, inactive tabs, sidebars
- =surface=: low-contrast background atop base, panels, inputs
- =overlay=: medium-contrast elevated background, popups, dialogs, active tabs
- =muted=: low-contrast foreground, comments, disabled text, ignored items
- =subtle=: medium-contrast foreground, punctuation, secondary labels, operators
- =text=: high-contrast foreground, main readable content
- =love=: errors, destructive actions
- =gold=: warnings, caution states
- =rose=: warm accent, booleans, modified state
- =pine=: success-ish or green semantic role, functions, renamed state
- =foam=: informational accent, additions, keys
- =iris=: links, focus, hints, active accent
- =highlight-low=: cursorline or low-emphasis selection background
- =highlight-med=: standard selection background
- =highlight-high=: borders, separators, stronger emphasis

* Default Mapping
Use these semantic mappings when converting an arbitrary theme to Rosé Pine Moon.

- background -> base
- panel -> surface
- popup -> overlay
- primary-text -> text
- secondary-text -> subtle
- disabled-text -> muted
- error -> love
- warning -> gold
- success -> pine
- info -> foam
- link -> iris
- focus-ring -> iris
- selection -> highlight-med
- border -> highlight-high
- cursorline -> highlight-low
